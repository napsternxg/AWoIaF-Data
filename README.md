# Exporting data from A Wiki of Ice and Fire

Collect data from http://awoiaf.westeros.org/index.php?title=Category:A_Song_of_Ice_And_Fire_chapters

## Dependencies
* Python 3
* BeautifulSoup
* requests

## Usage

```
python export_awoiaf.py
```
